# Axiom Agent

An autonomous AI operations agent that:
- Ingests leads
- Analyzes intent with OpenAI
- Drafts follow-ups
- Creates Notion tasks
- Persists data in SQLite
- Provides an Ops dashboard

## Stack
- Next.js (App Router)
- SQLite (better-sqlite3)
- OpenAI API
- Notion API

## Status
MVP complete. Expanding dashboard + multi-agent support.
